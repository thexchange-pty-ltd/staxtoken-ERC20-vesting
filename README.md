# staxtoken-ERC20-vesting
StaXtoken: ERC20 and nested vesting contracts

## Copy env

```sh
cp .env.example .env
```
## Install dependecies

```sh
npm install
```

## Compile smart contract

```sh
npm run compile
```

## Deploy smart contract

```sh
npm run deploy:mainnet #mainnet
npm run deploy:sepolia #testnet
```
